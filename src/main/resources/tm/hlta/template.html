IJCAI/AAAI (2000-15) Topic Tree

Instructions

  1. Click on the arrow on the left to collapse/expand a topic hierarchy.
  2. Click on a topic to show the documents belonging to that topic and the counts by year.
  3. Use the "Show Levels" button to select the levels (top-bottom) that you want to show.
  4. Use the "Filter" button to show only topics that contain the keyword along with their acestors and decendents. Use the "Clear" button to show all topics.
